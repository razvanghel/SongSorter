
## Getting Started

1. Set your artists directory's path in Settings.dart
2. Make sure your artists directory has the following structure
###    artistsDirectory ---GenreA---ArtistA
###                     |        |--ArtistB
###                     |--GenreB---SubgenreA---ArtistC
###                              |           |--ArtistD
###                              |--ArtistE
###                              |--ArtistF
